About Me

Haloo, aku Ahmad Fashich Azzuhri Ramadhani, yang akrab disapa Fashich, aku adalah seorang Junior Full-Stack Developer dengan tahap pengembangan dalam merancang, mengembangkan, dan mengoptimalkan solusi teknologi secara bertahap. Saat ini aku sedang menempuh Semester 4 program S1 Sistem Informasi di Universitas Negeri Surabaya, aku memiliki kemampuan multidisiplin yang mencakup seluruh spektrum pengembangan perangkat lunak, mulai dari front-end yang responsif hingga back-end yang robust, serta integrasi sistem kompleks dengan berbagai bahasa pemrograman dan teknologi modern. Dengan penguasaan mendalam atas bahasa seperti JavaScript (untuk web dan mobile), Python (untuk data processing dan machine learning pipelines), PHP dan Ruby (untuk server-side scripting ), React.js dan Flutter (untuk cross-platform application), serta C++, C#, Java , dan Go (untuk high-performance computing dan system programming), dan mampu mengadaptasi pendekatan teknis sesuai kebutuhan proyek.

Portofolioku ini mencakup pengembangan aplikasi enterprise-grade, sistem real-time, arsitektur microservices, dan solusi berbasis cloud dengan penerapan best practices seperti clean code, test-driven development (TDD), dan CI/CD pipelines. Aku juga memiliki keahlian dalam mengoperasikan berbagai Integrated Development Environments (IDEs) seperti VS Code, Visual Studio, PyCharm, dan Apache NetBeans, serta alat-alat spesialis seperti Docker untuk containerization, Postman untuk API testing, Git untuk version control, dan MATLAB untuk simulasi teknik dan analisis data. Kemampuanku tidak hanya terbatas pada ranah software development konvensional, tetapi juga mencakup eksplorasi teknologi niche seperti pengembangan game dengan Roblox Studio, pemrograman jaringan menggunakan Cisco Packet Tracer, menganalisis jaringan internet mengunakan Wireshark dan prototipe IoT dengan Arduino/Raspberry Pi.
